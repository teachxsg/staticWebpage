Syllabus

Basic Syntax and Data Types

In this module, you'll learn the fundamental syntax and data types of Python, including strings, integers, floats, and booleans.

Variables and Operators

In this module, you'll learn how to create and manipulate variables, as well as how to use operators to perform arithmetic and logical operations.

Conditional Arguments

In this module, you'll learn how to use conditional statements to control the flow of your program, including if/else statements and ternary operators.

Control Structures

In this module, you'll learn about control structures such as loops and iterators, which allow you to repeat actions and work with collections of data.

Functions

In this module, you'll learn how to create and use functions to organize and reuse your code, as well as how to work with function arguments and return values.

Lists and Dictionaries

In this module, you'll learn how to work with lists and dictionaries, two powerful data types in Python that allow you to store and manipulate collections of data.

Importing Modules and Working with Files

In this module, you'll learn how to import modules and use them in your programs, as well as how to read and write data to files.

Projects

In this module, you'll have the opportunity to apply what you've learned and work on practical projects, such as building a Telegram bot and a weather app.
